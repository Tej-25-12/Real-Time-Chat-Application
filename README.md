# Real-Time Chat Application

A full-stack real-time chat app built with **React (client)** and **Node.js + WebSocket (server)**.

## Features
- Real-time messaging between multiple users
- WhatsApp-style chat bubbles and contacts panel
- Enter-to-send functionality
- Backend with Node.js, Express, and WebSocket
- MongoDB integration for storing chat history
